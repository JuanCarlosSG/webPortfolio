Panteras Coding

Proyecto Estudiantil

Panteras Coding es un proyecto estudiantil que tiene como objetivo aprender y reforzar los conocimientos de diferentes lenguajes de programación. Este grupo forma parte de Grupos Panteras de la Universidad Panamericana. Además, busca aplicar todos esos elementos en proyectos internos y externos de alta magnitud. Soy el líder de este proyecto y coordinador de manera directa el area de desarrollo móvil.

Panteras Project Management

Aplicación

Panteras Project Management es un proyecto elaborado por el equipo de Panteras Coding de la Universidad Panamericana. Este proyecto tiene como objetivo administrar los diferentes equipos estudiantiles, incluyendo sus objetivos, miembros y proyectos. En este caso, formo parte del equipo de desarrollo de la aplicación para dispositivos móviles y también soy el coordinador del proyecto en general.

YoYo Mobility

Aplicación

YoYo Mobility es un startup que tiene como objetivo centralizar todos los servicios de movilidad dentro de la república mexicana. En este proyecto, tome el rol de desarrollador principal para la plataforma iOS en dispositivos móviles. El proyecto está programado en Swift, utilizando la versión de SwiftUI.
